## Script for mapping(mouse)
The script map_mouse_script (bash) downloads the fastq file in order to perform quality control. As a result produce the mapping and counts how many reads map to each feature (genomic feature). 

## Script for download data from NCBI
The script down_covid.sh downloads fastq file from NCBI with the command:

```
fastq-dump --split-files SRR13448536

```
